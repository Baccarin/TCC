SOAP


Simple Object Access Protocol (Protocolo de Acesso a Objetos Simples).

Cliente e servidor trocam mensagens usando XML.

Define mais padrões que carregam mais informações.

Pontos positivos:

  • Lógica de sucesso: Possui uma lógica de repetição integrada que proporciona uma alta confiabilidade de ponta a ponta.

  • Conformidade ACID: Tal conformidade reduz a ocorrência de erros e oferece um melhor funcionamento do banco de dados, mantendo a sua integridade, por meio de uma prescrição de como as transações podem interagir com o banco de dados.

  • Sendo o modelo ACID considerado um dos mais conservadores entre seus concorrentes, por este motivo, é muito utilizado em operações que transmitem informações confidenciais, assim como transações financeiras.

  • Segurança robusta: Além do suporte SSL, o WS-Security é um padrão integrado que permite que o SOAP ofereça mais alguns recursos de segurança de nível empresarial, caso você precise deles.
