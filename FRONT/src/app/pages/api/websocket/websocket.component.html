WebSocket


Usa objetos JSON para comunicação.

Oferece suporte à comunicação bidirecional entre aplicativos cliente e o servidor.

Servidor enviar mensagens de retorno de chamada a clientes conectados.

Executado em browsers e servidores web.

WebSocket define dois tipos de esquemas de URL:

  • WS: Para conexões não criptografadas.
  • WSS: Para conexões criptogradas.
